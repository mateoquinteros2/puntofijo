¿Cómo insertar una función?

En el primer campo se le pide que ingrese su f(x).
Esto lo hace para poder visualizarla en la gráfica.

En el segundo campo se le pide ingresar su g(x).
Una vez ingresado el programa calculará su derivada, en caso que la derivada evaluada en la aproximación ingresada sea mayor que 1, se mostrara un mensaje en pantalla que le dira que no existe raíz

USO DE ABREVIACIONES MAS COMUNES:

Si desea ingresar la función coseno debe escrbir cos(x)
Si desea ingresar la función seno debe escrbir sin(x)
Si desea ingresar la función tangente debe escrbir tan(x)
Si desea ingresar Logaritmo Natural debe escribir log(x)
Si desea ingredar el Número de Euler debe escribr e
Si desea ingredar el Número PI debe escribr pi
El valor de X dentro de las funciones anteriores puede ser sustituido por cualquier número o expresión.
Si desea ingresar un exponente debe escribir ^ donde lo anterior a ^ el es la base y lo siguiente es el exponente.

Recuerde usar paréntesis para agrupar expresiones.
Si desea escribir una funcion como 2cos(x) ó 3x. Se recomienda usar * para denotar la multiplicación, quedando 2*cos(x) y 3*x

¿Cómo insertar la aproximación?

En el tercer campo se le pide el número inicial de X, puede ser un número positivo o negativo. No se puede usar / para indicar fracciones, se recomienda usar . (punto decimal)
